# ☁️ 100 Days of Cloud Computing

Welcome to my daily learning repository! This project is a commitment to learning and sharing small, digestible pieces of Cloud Computing knowledge every single day.[cite: 6]

## 🚀 The Goal
The objective is to build a consistent habit of learning. Each day, I will add a new file containing detailed notes (minimum 30 lines of technical content) about a specific cloud concept, starting from the absolute basics and moving toward advanced architecture.

## 📁 Repository Structure
Each day's learning is documented in its own markdown file:
- `day01_intro_to_cloud.md`
- `day02_deployment_models.md`
- ... and so on.

## 📖 Curriculum Overview
1. **Basics:** Definitions, Characteristics, Benefits.
2. **Models:** IaaS vs PaaS vs SaaS, Public vs Private vs Hybrid.
3. **Compute:** Virtual Machines, Containers, Serverless.
4. **Storage:** Block, Object, and File storage.
5. **Networking:** VPCs, Subnets, DNS, Content Delivery Networks.
6. **Security:** Identity and Access Management (IAM), Encryption.
7. **Providers:** Deep dives into AWS, Azure, and Google Cloud.
